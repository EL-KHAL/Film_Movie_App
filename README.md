## Run Locally

Clone the project

```bash
  git clone https://github.com/EL-KHAL/film_movie_app.git
```

Go to the project directory

```bash
  cd film_movie_app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start ya khali
```
